# Document Writer Skill

Use this skill when creating Word documents, business documents, architecture documents, SOPs, whitepapers, proposals, implementation guides, or executive summaries.

Rules:
- Produce clean Markdown.
- Start with an executive summary.
- Use clear headings and subheadings.
- Tie claims to provided context only.
- Do not invent facts.
- State assumptions explicitly.
- Include tables where useful.
- Write for senior technical and business stakeholders.
